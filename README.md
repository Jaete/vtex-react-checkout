# React Checkout Starter Template (VTEX)

Este é um starter pack completo e estruturado para desenvolvimento de Checkout VTEX utilizando React. Ele oferece uma base robusta com gestão de estado, hooks customizados, integrações com VTEX IO e uma arquitetura modular.

## 🚀 Começando

### Requisitos

- Node.js (Recomendado: LTS)
- npm ou yarn
- VTEX IO CLI ([Instalação](https://developers.vtex.com/docs/guides/vtex-io-sdk-cli#install-the-cli))

## 🤝 Contribuindo

Sinta-se à vontade para abrir issues e pull requests. Para contribuições:

1. Faça um fork do projeto.
2. Crie uma branch para sua feature (`git checkout -b feature/NovaFuncionalidade`).
3. Commite suas mudanças (`git commit -m 'feat: Adiciona nova funcionalidade'`).
4. Push para a branch (`git push origin feature/NovaFuncionalidade`).
5. Abra um Pull Request.
